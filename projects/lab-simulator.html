Chemical Engineering Laboratory Simulator

Fully Functional

A comprehensive virtual laboratory application for chemical engineering students to simulate, learn, and practice various experiments without physical lab access. This platform combines interactive simulations, educational content, and modern web technologies to deliver an immersive learning experience.

Project Overview

The Chemical Engineering Laboratory Simulator is a comprehensive educational platform designed to provide students with a virtual laboratory experience that closely mimics real-world experiments. It addresses the challenges of limited lab access while offering enhanced learning opportunities through interactive simulations and guided learning modules.

Python-based backend with Streamlit and FastAPI
Modern React frontend for interactive UI
Mobile-responsive design with dark/light themes
Real-time data visualization and simulation

Key Features

Interactive Simulations

10 chemical engineering experiments with adjustable parameters and real-time visualization of process dynamics and results.

Demo Videos

Professionally recorded demonstration videos showing actual laboratory procedures and equipment operation for each experiment.

Educational Quizzes

Interactive quizzes with multiple choice and numerical questions to test understanding of experiment principles and procedures.

Report Generation

Automated creation of professional laboratory reports with simulation data, graphs, and template-based formatting in DOCX format.

Chat Assistant

AI-powered chat interface providing instant answers to questions about chemical engineering concepts, experiment procedures, and calculations.

Mobile Support

Responsive design optimized for mobile devices with native app build support for Android, providing on-the-go access to all features.

Experiment Modules

Reactor Experiments

  • • Isothermal Batch Reactor
  • • Isothermal Semi-batch Reactor
  • • Isothermal CSTR
  • • Isothermal PFR

Model kinetics, concentration profiles, and conversion in various reactor configurations while adjusting key operating parameters.

Separation Processes

  • • Plate and Frame Filter Press
  • • Rotary Vacuum Filter
  • • Centrifuge and Flotation
  • • Classifiers

Simulate solid-liquid separation operations with visualization of filtration rates, cake formation, and separation efficiency.

Size Reduction

  • • Crushers and Ball Mill
  • • Trommel

Study particle size reduction, energy requirements, and product size distribution in common comminution equipment.

Technical Implementation

Architecture

├── app.py                  # Main application entry point
├── chemengsim/             # Core application package
│   ├── experiments/        # Simulation modules for each experiment
│   ├── videos/             # Demo video handling module
│   ├── quizzes/            # Quiz system and question data
│   ├── report_generation/  # Lab report generation system
│   ├── chat.py             # Chat assistant implementation
│   ├── api.py              # FastAPI backend for React UI
│   └── react_src/          # React frontend components
└── requirements.txt        # Python dependencies

The application follows a modular architecture where each experiment is implemented as a separate module with consistent interfaces for simulation, visualization, and interactivity.

Backend Technologies

  • • Python 3.8+: Core programming language
  • • Streamlit: Main web application framework
  • • FastAPI: API backend for React communication
  • • Pandas & NumPy: Data processing and numerical computation
  • • Matplotlib & Plotly: Data visualization and plotting
  • • SciPy: Scientific computing and integration
  • • Python-DOCX: Report generation in Word format

Frontend Technologies

  • • React: Modern UI framework
  • • Styled Components: CSS-in-JS styling
  • • React Router: Client-side navigation
  • • WebSockets: Real-time data streaming
  • • SVG/Canvas: Interactive visualizations
  • • Responsive Design: Mobile-optimized layouts
  • • Theme Support: Dark/light mode interfaces
