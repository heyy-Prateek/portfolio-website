Hartree-Fock Quantum Chemistry Solver

Active Development

A modular, high-performance implementation of Restricted and Unrestricted Hartree-Fock methods in C++. This program calculates electronic structures and energies of molecules using Gaussian basis functions and advanced SCF convergence techniques.

Architecture Overview

The program follows a modular, component-based architecture with separate libraries handling different aspects of quantum chemistry calculations:

  • libscf
    Self-Consistent Field solvers (RHF/UHF) with DIIS and fixed-point iteration methods
  • libint
    One and two-electron integral calculations with optimized Gaussian primitives
  • libmol
    Molecule representation, basis set management, and geometry handling
  • libmath
    Linear algebra utilities including matrix diagonalization and eigenvalue solvers
  • libgf
    Gaussian function primitives and normalization for quantum chemistry

Key Features

Multiple Methods

Support for both Restricted (RHF) and Unrestricted (UHF) Hartree-Fock calculations to handle both closed and open-shell systems.

Theoretical Background

The Hartree-Fock method is a fundamental quantum chemistry approach that approximates the electronic wavefunction and energy of a quantum many-body system.

$$F\psi_i = \varepsilon_i \psi_i$$

Hartree-Fock equation, where \(F\) is the Fock operator, \(\psi_i\) is a molecular orbital, and \(\varepsilon_i\) is the orbital energy.

The Fock operator contains kinetic energy, electron-nuclear attraction, and a mean-field electron-electron repulsion:

$$F = -\frac{1}{2}\nabla^2 - \sum_A \frac{Z_A}{r_{iA}} + V_{HF}(i)$$

Where \(V_{HF}\) represents the average potential an electron experiences due to all other electrons.

The SCF procedure is implemented with both fixed-point iterations and DIIS for enhanced convergence:

// Pseudocode for the SCF procedure in RHF
void SCF_RHF() {
    // Initial guess for density matrix P
    P = initial_guess();
    
    do {
        // Build Fock matrix from density matrix
        F = build_fock_matrix(P);
        
        // Solve F*C = S*C*E eigenvalue problem
        C = solve_generalized_eigenvalue(F, S);
        
        // Form new density matrix
        P_new = form_density_matrix(C);
        
        // Check for convergence
        error = calculate_error(P, P_new);
        P = P_new;
        
    } while (error > threshold);
    
    // Calculate final energy and properties
    energy = calculate_energy(P, F, H_core);
}

The implementation also includes specialized routines for UHF to handle open-shell systems with different alpha and beta electron densities.

Key Implementation Details

SCF Convergence with DIIS

The Direct Inversion in the Iterative Subspace (DIIS) method drastically improves convergence by using error vectors from previous iterations to predict an improved Fock matrix.

Efficient Integral Evaluation

Optimized algorithms for computing one- and two-electron integrals using the Gaussian product theorem and recurrence relations.

Population Analysis

Both Mulliken and Löwdin methods for analyzing the electron distribution and atomic charges in molecules.

Memory Management

Efficient storage and manipulation of large matrices for two-electron integrals and density matrices.
